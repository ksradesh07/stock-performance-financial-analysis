# Stock Performance & Financial Analysis (Python):

## Overview of My Project 
This project analyzes stock performance and company financials using 
the coding language python. It compares stock returns and evaluates Apple's revenue 
trends using real market and financial statement data. 

## Objectives of this project
- Compared stock performances of Apple, Microsoft, and Tesla.
- Normalized stock prices in order to show the growth of a $100 investment.
- Analyzed Apple's annual revenue trends.

## Tools used
- Python
- yfinance
- pandas
- matplotlib
- numpy

## Insights
- Tesla displayed higher cumulative returns which reflected high volatility.
- Apple's revenue was relatively stable year-over-year.
- Revenue trends and stock performance do no always move together.
